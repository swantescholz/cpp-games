ziel: man muss zum haus kommen und danach tiefer sein als der tiefste feste stein
	auf dem rueckweg kann man dabei weder spingen noch steine setzten


steurung:
-> <- und linke maustaste = links rechts
b und rechts maustaste = jump
n = stein setzten
m = stein aetzen
f = fullscreen und zurueck
t = zum letzten checkpoint
RETURN(ugs.: [ENTER]) = pause


im letzten level sieht man einen geheimcode
gibt man ihn ein wird freigeschaltet:

- v = wechsel zum ball und zurueck
- c = next level
- x = last level

-e = editor und zurueck:
w/s = load level up/down
a/d = save level up/down
-> <- und mausrad = blockart up/down
q = modus up (0 = frei, 1 = reihen legen, 2 = rasterfoermig)
g = save

nur fuer wolken:
i/k = richtung up/down (1 = N; 2 = NO; ... 8 = NW)
u/o = pixel in eine richtung up/down
j/l = pixel in die andere richtung up/down
7/8 = speed up/down

hoch/runter = anzahl der dazwischenliegenden steine bei modus = 1 up/down

linke maustaste = stein setzten
rechte maustaste = stein loeschen

7,8,9,6,3,2,1,4 = map rasterfoermig verschieben
