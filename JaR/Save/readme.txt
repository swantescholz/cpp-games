readme zu JaR 2008 by Eric (C.)

Ziel des spiels ist es zu einem haus zu kommen und dann wieder zurueck auf den boden.
b = jump; n = stein setzen; m = stein wegnehmen; f = zwischen fullscreen und fenster mde wechseln; c = naechstes level; x = vorheriges level;
escape = menue;

-im Save-ordner:
	allgemein : immer mit 2 doppelkreuzen abschliessen;darin kann ein kommentar stehen
	-im kofigurationstext kann man bestimmte werte aendern
	-man kann die levelordner veraendern oder neu erstellen; aber dann mit koorrekter syntax;
		-es werden immer reihen gelegt; es gilt:
			-1.wert: art des blocks: 0 = fest, 1 = gestrichelt(lieber nicht verwenden), 2 = wegnehmbar, 3=ziel
			-2. und 3. wert: x und y wert des ersten blocks
			-4. wert: anzahl der bloecke zwischen 1. und 2. block der aktuellen reihe
			-5. und 6. wert: x und y koordinaten des zweiten blocks
			-7. : falls art = 1: 0:wenn er nach einer sekunde verschwinden soll, 1 = wenn er nach einer sekunde zum block der art
					2 werden soll;; ansonsten auch immer 0!!!;

-viel spass
